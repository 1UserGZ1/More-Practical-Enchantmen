# More Practical Enchantments Mod Introduction (Simplified Version)
## Mod Basic Info
Name: More Practical Enchantments
Version Support: Minecraft Java 1.20.1 (Forge 47.x.x)
Author: DY_XiaoJiaZi
Type: Vanilla Enhancement / Enchantment Expansion
Languages: Full English & Chinese Localization
Brief: Adds 13 brand new balanced vanilla-style enchantments with configurable values and unique mechanics.

## Mod Features
1. 13 new enchantments covering weapons, armors, tools and elytra
2. Well-balanced tier system, exclusive conflicts and drawbacks to avoid overpowered builds
3. Synergies and trade-offs between enchantments for strategic gameplay
4. Full config file to tweak all enchantment statistics
5. Dedicated creative tab for easy enchantment book testing
6. Complete bilingual localization

## Custom Configuration
Config File: `morepracticalenchantments-common.toml`
Tweakable values: Lifesteal percentage, magma damage reduction, attack speed bonus, XP multiplier, sprint speed boost, Natural Mending cooldowns, emerald drop amounts and all other enchantment stats.
